Performing ETL on Medicare Data with SQLAlchemy

This project demonstrates the core workflow for ETL (Extract / Transform / Load), using a Medicare dataset of Quality Incenitve Program Scores for End-Stage Renal Disease clinics.


The main tasks:
  • - Connect Jupyter notebook to a SQLite database
  • - Extract data with SQL queries
  • - Transform data with Pandas
  • - Load data to csv files


Steps:
  • Import python dependencies


  • Create the engine and connect to the database


  • Extract the data via SQL queries


  • Transform the query results into pandas dataframes


  • Load the data into csv files





In phase 2 of this project, I will use the loaded csv files to build an interactive visualization with javascript & D3.


See the full Jupyter notebook on my GitHub.
