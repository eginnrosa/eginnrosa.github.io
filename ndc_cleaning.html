  • Code: Repo on GitHub

Combining and cleaning two datasets using python, pandas, and openpyxl

The main tasks:
  • Merge two related NDC datasets into one dataframe
  • Clean the data for better analysis/visualization
  • Write cleaned dataframes to Excel as an aggregate file of summary tables



Dirtiness includes:
  • Duplicate columns
  • Mixed case cell values
  • Dates stored as numbers (outlined below)
  • Typos resulting in varied cell values that should be the same (highlighted below)



DataFrames created and exported to Excel:
  • Count of drugs by drug type
  • Count of drugs by route
  • Count of drugs by marketing category
  • Count of drugs by labeler
  • Count of drugs by marketing start year



The exported Excel tables can be utilized for presentation, visualization, or further trend analysis.
